Sobre C

C é uma linguagem de programação compilada de propósito geral, estruturada, imperativa, procedural

padronizada pela Organização Internacional para Padronização (ISO)

criada em 1972 por Dennis Ritchie na empresa AT&T Bell Labs

para desenvolvimento do sistema operacional Unix (originalmente escrito em Assembly).

Usos de C

  • Desenlolvimento de compiladores
  • Desenvolvimento de sistemas operativos para desktop ou smartphone
  • Desenvolvimento e construção de bases de dados
  • Desenvolvimento de browsers e de extensões
  • Desenvolvimento de aplicações creativas, como ferramentas de fotografia e design grafico

O criador de C

Dennis MacAlistair Ritchie foi um enginheiro informático americano.

Criou a linguagem de programação C e com o seu colega de longa data Ken Thompson, o sistema operativo Unix operating.

Ritchie e Thompson foram premiados com o Turing Award da associação de Computing Machinery (ACM) em 1983.

Medalha Hamming do instituto de eletricidade e enginharia eletrónica(IEEE) em 1990.

Medalha nacional de tecnologia dada pelo presidente Bill Clinton em 1999.
