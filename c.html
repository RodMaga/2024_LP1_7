The creator of C

C é uma linguagem de programação compilada de propósito geral, estruturada, imperativa, procedural, padronizada pela Organização Internacional para Padronização (ISO)

criada em 1972 por Dennis Ritchie na empresa AT&T Bell Labs

para desenvolvimento do sistema operacional Unix (originalmente escrito em Assembly).

The uses of C

  • Developing compilers
  • Developing the operating systems used on a desktop or smartphone
  • Developing and building databases
  • Developing internet browsers and their extensions
  • Developing creative applications, such as photography and graphic design tools along with desktop and system applications
  • Working in embedded systems
